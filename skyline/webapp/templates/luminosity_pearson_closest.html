Pearson :: closest

You can analyse a metric against other metrics to determine the metrics that correlate closest to the metric over a time period.

Option value
Pearson closest
Metric the metric to correlate
Namespaces a comma separated list of metric/namespace patterns you want to correlate the metric with
from_timestamp (unix timestamp or %Y-%m-%d %H:%M)
until_timestamp (unix timestamp or %Y-%m-%d %H:%M)
abs_threshold the absolute correlation threshold to use, any correlations under this value will not be returned a float between 0.0 and 1.0
1.0 being a perfect correlation and 0.0 being no correlation, consider anything > 0.8 as starting to describe a strong correlation.
max_results the maximum number of results to return, 0 returns all. If a value is set the truncated results will be ordered by strongest to weakest correlation.
plot_graphs
Output format default output is to the Luminosity page, if you want a json response set this to json.
