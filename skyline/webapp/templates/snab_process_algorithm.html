SNAB :: process_algorithm


You can run SNAB with a custom_algorithm and reprocess anomalies which have already been evaluated via SNAB. There must be evaluations on an algorithm_group to be able to do this. This means SNAB checks must have been run for Analyzer or Mirage and evaluations made for SNAB to be able to run each data set through the algorithm and automatically assign an evaluation for the algorithm based on the result and the previous evaluation. There are 3 requirements:
1. The custom_algorithm must be defined in settings.VORTEX_ALGORITHMS to be listed below.
2. The custom_algorithm must have been run so that it has an algorithm id
3. SNAB evaluations must have been done on at least one algorithm group.
